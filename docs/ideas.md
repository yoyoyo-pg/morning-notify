### 🔥 松岡修造（タスクリマインダー）

**役割**: 昼にNotionの未完了タスクを激励つきで通知する  
**タイミング**: 毎日 12:00  
**データソース**: Notion API（当日ジャーナルの未完了チェックボックス）

```
🔥 まだやれる！絶対できる！諦めるな！

今日やるって決めたこと、まだ 3個 残ってるぞ！
太陽より熱くなれ！！！

📋 未完了タスク
  ☐ 企画書レビュー
  ☐ テストコード書く
  ☐ 請求書送る

日本一熱い男より愛を込めて 🔥🔥🔥
```

**実装メモ**
- `matsuoka/` ディレクトリで管理（ショーンK・厚切りジェイソンと同様に分離）
- Notion API で当日ジャーナルの未完了タスクを取得
- タスク0件のときは「全部やり切ったな！俺は感動した！！」的なメッセージを出す

### 💛 アンミカ（月曜モチベ）

**役割**: 月曜の朝だけ流れる自己肯定・週次モチベ通知  
**タイミング**: 月曜 6:00（ショーンKの直後）  
**データソース**: メッセージは数パターン用意してランダム表示

```
💛 おはようさん！5/6(月)やで！

あんたほんまに頑張ってるんやから、
今週も自分を信じてあげてな。

✨ 今週意識してほしいこと
「完璧じゃなくていい。昨日より0.1%だけ前に進めばええねん」

あんたが幸せでいることが、周りも幸せにするんやで 💛
```

**実装メモ**
- `anmika/` ディレクトリで管理
- 月曜のみ実行するcron（`0 21 * * 1` UTC）
- 励ましメッセージをリストで管理してランダム選択

### 🐟 さかなクン（技術ニュース専門）

**役割**: Zennトレンドをさかなクン口調で別チャンネルに流す  
**タイミング**: 毎日 6:00（ショーンKと同時、別チャンネル）  
**データソース**: Zenn RSS（現在ショーンKのニュース枠に含まれているものを分離）

```
🐟 ギョギョ！今日の技術トレンドだよ〜！

【Zennより3件ギョ！】
・TypeScript 5.8の新機能がすごいギョ！
  https://zenn.dev/...
・Rustで作るWebアセンブリ入門
  https://zenn.dev/...
・Claude APIのtool useを使いこなす
  https://zenn.dev/...

さかなクンはこの記事が特に気になるギョ〜！🐡
```

- ショーンKのニュース枠からZennを除くかどうかは要検討

### 😱 出川哲朗（エラー・障害アラート）

**役割**: GitHub Actions の失敗時にだけ発火するアラートボット  
**タイミング**: ワークフロー失敗時のみ（定期実行ではない）  
**データソース**: GitHub Actions のワークフロー失敗イベント

```
😱 やばいよやばいよやばいよ！！

morning-notify.yml が失敗しちゃってるよ！！
ショーンKからの朝の通知が届いてないかも！！

❌ エラー内容
  DISCORD_WEBHOOK_URL が設定されてないっぽいよ！！

早く直してよ〜〜頼むよ〜〜！！ 😱😱😱
```

**実装メモ**
- 既存ワークフローに `on: workflow_run` トリガーを追加する形で実装
- 専用ディレクトリではなく `.github/workflows/alert-notify.yml` 単体で完結できるかも

### 💄 マツコ・デラックス（週次レビュー）

**役割**: 金曜夜に1週間を辛口でまとめる週次振り返り通知  
**タイミング**: 金曜 21:00（ショーンKの夜通知の代わりに週1だけ発火）  
**データソース**: Notion API（今週のジャーナルの完了・未完了タスク集計）

```
💄 あら、今週も終わりね。

📊 今週（4/28〜5/2）の振り返り

完了: 8件 ／ 未完了: 3件

未完了のやつ、来週こそやんなさいよ。
あなたにはできるって私は思ってるけど、
本当にやる気あるのかしらね。

まあ、お疲れ様。ゆっくり休みなさい。💄
```

**実装メモ**
- `matsuko/` ディレクトリで管理
- 金曜のみ実行するcron（`0 12 * * 5` UTC = 金曜21時JST）
- Notion APIで今週分のジャーナルページを集計（タスクの完了率を出す）
- Notion未連携の場合は固定メッセージのみ

### 💪 ジョージ（朝の男前通知）【迷い中】

**役割**: 朝のショーンKの代替候補。静かな圧と断定で一日を始める  
**タイミング**: 毎日 6:00（ショーンKと役割を入れ替える形）  
**データソース**: ショーンKと同様（天気・スケジュール・ニュース）

```
💪 起きろ！5/1(木)だ！

男は言い訳をしない。
今日やるべきことをやれ。それだけだ。

📅 本日のスケジュール
  ・10:00 週次MTG
  ・14:00 1on1

お前が動かなければ、誰も動かない。
今すぐ始めろ。💪
```

**実装メモ**
- `src/reminkun/` のコードはそのまま流用し、文言だけ変える
- 松岡修造との違い：修造は「熱量で押す」、ジョージは「断定と静圧」
- ショーンKを夜専任にする場合、朝担当としてジョージを立てる選択肢がある
